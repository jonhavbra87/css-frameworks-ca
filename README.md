## Getting Started

Download form the css-frameworks branch

### Installing

Clone the repo:

```bash
git clone https://github.com/jonhavbra87/css-frameworks-ca.git
```

### Running

Open in VSCode.
Open terminal.
Write:

```bash
npm install
```

Then in the same terminal:

```bash
npm run dev
```
